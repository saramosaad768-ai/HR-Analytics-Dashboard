# 👥 HR Analytics Dashboard

## 📝 Project Overview
This project focuses on analyzing Human Resources data to help organizations understand employee trends. The dashboard tracks key metrics like attrition rates, employee satisfaction, and performance distribution.

## 🛠️ Tools & Skills
* **Power BI:** Data visualization & storytelling.
* **Power Query:** ETL and data cleaning.
* **DAX:** Calculated measures for HR KPIs.
* **People Analytics:** Understanding workforce dynamics.

## 📊 Key KPIs Analyzed
* **Attrition Rate:** Identifying why employees leave.
* **Performance vs. Salary:** Analyzing the correlation between pay and output.
* **Demographics:** Workforce distribution by department, age, and gender.
